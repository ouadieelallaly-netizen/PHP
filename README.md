# Quiz PHP simple

## Description
Petite application de quiz réalisée en PHP sans base de données.

## Contraintes respectées
- Pas de programmation orientée objet
- 100% local
- Aucun framework
- Fichiers PHP simples

## Fichiers
- index.php : questions du quiz
- result.php : calcul et affichage du score

## Lancement
- Placer le dossier dans htdocs (XAMPP)
- Lancer Apache
- Ouvrir http://localhost/quiz-php/index.php
